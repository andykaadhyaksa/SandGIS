📍 Upload KMZ / KML
Unggah file .kmz (Google Earth, paket terkompresi) atau .kml (XML terbuka) untuk ditampilkan sebagai layer di peta. Titik, garis, dan poligon dari Placemark KML akan dirender otomatis. Koordinat menggunakan WGS84 (Lon/Lat).

🗄️ Upload GeoPackage
Unggah file .gpkg (GeoPackage — SQLite berisi banyak layer geospasial) untuk ditampilkan sebagai layer di peta. Setiap tabel/layer di dalam GeoPackage akan dimuat terpisah. Koordinat sumber diasumsikan WGS84 (EPSG:4326); layer dengan sistem koordinat lain sebaiknya direproject dulu di QGIS.

🏔️ Upload TIF (DEMNAS)
Unggah file .tif / .tiff GeoTIFF — misalnya DEM dari BIG DEMNAS (tanhair.big.go.id) — untuk ditampilkan sebagai layer elevasi di peta. Proyeksi WGS84 (EPSG:4326) paling kompatibel; file besar mungkin butuh waktu lebih lama. Catatan: layer TIF tidak ikut tersimpan di file project (.riverload) karena ukurannya besar — perlu diunggah ulang tiap sesi.
Drag & drop file .tif di sini
atau klik untuk pilih file

Seret untuk mengatur urutan. Prioritas teratas = menang di area overlap. Klik "Terapkan" untuk render mozaik gabungan.

📐 Upload DXF
Unggah file CAD .dxf berkoordinat (mis. hasil digitasi AutoCAD/QGIS) untuk ditampilkan sebagai layer garis/poligon di peta. Koordinat DXF harus sudah dalam sistem geografis (Lon/Lat, WGS84) atau UTM — pilih zona UTM di bawah kalau file kamu memakai koordinat UTM (meter).
Sistem koordinat file DXF
Zona UTM (Indonesia: 46–54)
